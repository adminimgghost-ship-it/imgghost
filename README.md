# ImgGhost

ImgGhost is a secure private image sharing platform that lets you share photos safely without exposing them publicly.

## Features

- Password protected image sharing
- Auto delete after selected views
- View limits
- Automatic image expiration
- Fast and secure uploads
- No registration required

## Why ImgGhost?

Public image links can remain accessible long after they are shared.

ImgGhost gives you better control by allowing you to:

- Share private screenshots securely
- Protect sensitive images with passwords
- Limit how many times an image can be viewed
- Automatically delete images after they are viewed
